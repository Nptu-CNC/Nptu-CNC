Vscode 快捷鍵

ctrl + alt + 方向鍵上(下) : 向上(下)新增游標
ctrl + D : 選取你所反白的內容
alt + 方向鍵上(下) : 垂直移動游標所處的該行
shift + alt + f : 自動排版
ctrl + shift + \ : 跳到適配的括號
ctrl + [ 或 ] : 進縮排
alt + 滑鼠點擊 : 被點擊的該位置可同時編輯
ctrl + alt + P :可叫出Command Palette
