## Parcel
Parcel是一款免設定且方便使用的打包工具，
跟webpack相比整體上便利很多，
但也因為這樣的特性所以也少了很多彈性。

在使用Parcel之前需要先安裝將它安裝起來。
> npm install --save-dev parcel

使用的方式很簡單只要輸入以下指令即可執行：
> parcel your_Entry_Point_file
> 
> Example :
>
> parcel src/index.html
